# Files Overview

- diabetes_binary_5050split_health_indicators_BRFSS2015.csv  
  Dataset used for analysis

- main.py  
  Contains the code required to run the analysis

- Other files  
  Helper functions and environment setup files


# Instructions for Building and Running the Code

1. Install uv:  
   https://docs.astral.sh/uv/getting-started/installation/

2. Sync packages for repository by running:

   uv sync

3. Run main.py using the virtual environment:
   
   uv run main.py
